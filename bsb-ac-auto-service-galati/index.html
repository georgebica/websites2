// 01

Încărcare freon.
Aer rece imediat.

Lucrăm cu R134a şi R1234yf, ambele tipuri de freon utilizate pe parcul auto actual. Verificăm cantitatea exactă, scurgerile şi presiunea. Spunem totul înainte de încărcare.

  • Vidare circuit + verificare presiune.
  • Cantitate corectă de freon conform specificaţiei producătorului.
  • Test eficienţă — măsurăm temperatura de pe ventilaţie.
  • Raport scris cu valorile măsurate.
// 02

Igienizare A/C.
Aer curat.

Vaporizatorul A/C este un teren propice pentru bacterii şi mucegai. Igienizăm cu spray ULV, schimbăm filtrul de polen, eliminăm mirosul neplăcut din habitaclu.

Antibacterizare

Dezinfectare profundă a circuitului.

Filtru polen

Înlocuit la fiecare igienizare.

// 03

Verificare instalaţie completă.

Detecţie scurgeri

Cu trasor UV sau electronic. Localizăm exact pierderea, nu ghicim.

Test presiune

Verificare etanşeitate circuit pe partea de joasă şi înaltă presiune.

Schimb componente

Compresor, condensator, vaporizator, supape, conducte.
